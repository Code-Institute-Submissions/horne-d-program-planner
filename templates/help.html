Help

Find any help you need below, or tips on how to make the most of the application.

How can I add a new program?

You can add a new program by following the link - Add a Program

I can't find a suitable category, how do I add a new one?

You can add a new category by following this link - Add a Category

Can you make changes to a program after you add it?

Yes, when viewing the programs there is a pencil icon. Click this and you can make changes to the program.

I have added a program by mistake, what can I do?

You can either edit the program using the steps mentioned above, or you can delete the program. To delete, look for the trash icon when viewing the programs, and this will delete it from the list.

Do I need to add a picture when adding a program?

No this is not essential. You can add a program without the need to get an image URL.

Is there a limit to how many programs I can add?

No, you can add as many, or as few as you like.
